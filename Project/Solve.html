Sum of All Fears

Given a set array of numbers and a number K, entered by the user, this program is written with an algorithm that will determine whether any two numbers from the set list add up to K.
The set numbers for this exercise are [10,15,3,7].

Alternate!

In this alternate exercise, the program will allow the user to input their own values for both the array and the K value.
Again, the algorithm will be run to evaluate whether any two numbers from the entered array add up to K.
